# Evolve

The user wants to extract and manage session learnings. Invoke the evolve skill with mode: **$ARGUMENTS** (if empty, default to `analyze`).

Analyze session history for patterns, review existing learnings, or list active intelligence. Every learning must be confirmed by the user before persisting. Evidence before assertions.
