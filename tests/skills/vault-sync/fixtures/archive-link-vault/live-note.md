Links to an archived note by BARE BASENAME: [[archived-note]].

The bare form is load-bearing for the regression test. A pathed link such as
`[[90-archive/archived-note]]` resolves via the existsSync candidate branch even
when the register excludes archived notes, which would make the test vacuously
green.
